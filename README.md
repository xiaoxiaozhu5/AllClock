# AllClock

## Feature
* Normal time display
* Calender display
* Bignumber clock display
* Temperature display
* Time set(year,month,week,day,hour,min)
* Alarm set(total six separate alarm, individual control)
* Hourly chime(nine different mode)
* Backlight control(auto control & manual control)
